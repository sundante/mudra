# Mudra

Mudra is a local-first personal finance app built with Flutter. It uses
Supabase for identity, while financial records stay device-local in a
user-scoped Isar store.

## Current Shape

- Home has two tabs: `This Month` and `Overall`.
- `This Month` leads with the fuel gauge. The old gold Month Runway hero is
  intentionally removed.
- Gauge colour follows projected month-end value: grey at `0`, green when
  positive, red when negative.
- Profile is identity/settings-oriented: editable name, App Map, sign out,
  clear local data, and footer.
- App Map is available only from Profile and is rendered from
  `assets/maps/mudra_app_map.json`.
- `docs/vibes/APP_MAP.html` is generated by
  `dart run tool/generate_app_map_html.dart`.

## Useful Commands

flutter analyze
